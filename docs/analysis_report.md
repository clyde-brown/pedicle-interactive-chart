### 분석 방법

### 주요 패턴

#### 주증상 → 진단코드 빈출 조합

| 차팅 키워드 | 매핑 KCD | 진단명 |
|------------|----------|--------|
| "허리통증" | S3350 | 요추의 염좌 및 긴장 |
| "허리통증" + 다리 저림 | M4806 | 척추협착, 요추부 |
| "허리통증" + 다리 저림 | M511 | 신경뿌리병증 동반 추간판장애 |
| "무릎 통증" | M170 | 양쪽 원발성 무릎관절증 |
| "무릎 통증" | S800 | 무릎 타박상 |
| "발목 접지름" | S9348 | 발목 염좌 및 긴장 |
| "어깨 통증" + LOM | M750 | 동결견(오십견) |
| "손목 통증" + 건초염 | M6584 | 힘줄윤활막염, 손 |
| "허리 골절 과거력" | M4806 | 척추협착, 요추부 |

#### 차팅 내 임상 정보 구조 패턴
```
◈ 증상 ◈          → 주소증, 발병일, 기전
진단명 (영문)      → 의사 즉각 판단 (예: Acute lumbar sprain, Frozen shoulder Rt)
검사              → x-ray 소견, MRI 언급
치료 계획         → P-T, PO, SNRB, FU 기간
```

> **특이사항**: `symp_nm`이 구조화된 EMR 필드가 아닌 **자유텍스트**이므로, 영문/한글 혼용, 줄임말(P-T=물리치료, PO=경구약, FU=추적관찰) 다수 포함.

## 종합 인사이트

1. **핵심 진료 패턴**: 척추·관절 질환 → 초진(X-ray + 진찰) → 물리치료 3종 세트 → 필요시 신경차단술·주사 → 경구약(소염+위보호+근이완) 처방

2. **물리치료 집중도**: 상위 3개 처방이 모두 물리치료, 총 79,078건 처방 중 27% 차지

3. **재방문 집중**: 전체 내원의 66%가 7일 이내 재방문. 집중 통원 환자 다수

4. **차팅 품질**: `symp_nm`은 임상 정보가 풍부하나 자유텍스트라 NLP 전처리 필요. 영문 진단명 + 한글 설명 혼용 구조

5. **K219(식도역류) 급등**: 2024-02 이후 월 100건 이상. 소염제 다빈도 처방으로 인한 위장약 동반 처방 반영
