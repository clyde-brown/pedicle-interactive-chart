> ⚠️ 주의
> 
> - 아래 코드는 **교육/연구/개발(예: ICD‑11 API 테스트)** 용 예시입니다.
> - 실제 청구/통계/진료용으로 사용하기 전에 반드시
>     - WHO ICD‑10/ICD‑11 공식 브라우저,
>     - 병원 코드북,
>     - 국가 건보(심평원) 가이드
>     로 재확인해야 합니다.
> - **ICD‑11 코드는 직접 조회해서 채우도록 비워 두었습니다.**

컬럼:

- **번호**: 임의 번호
- **진단명(한글)**: 흔히 쓰는 정형외과 진단명
- **ICD‑10 코드**: 대표 코드 (국가/기관마다 세부 차이가 있을 수 있음)
- **ICD‑11 코드(직접 조회)**: WHO ICD‑11 브라우저·API로 조회해서 채우기

### 3. ICD‑11 코드를 채우는 방법 (프로젝트와 연계해서)

- **ICD‑11 브라우저(웹)**
    - WHO ICD‑11 브라우저에서
        - 진단명(영어) 또는 ICD‑10 코드를 검색
        - 해당 엔티티의 ICD‑11 코드 확인 후, 위 표의 “ICD‑11 코드” 열을 채우기
- **지금 사용 중인 ICD‑11 API 활용**
    1. ICD‑10 → ICD‑11 매핑 테이블에서 `icd11entity_id` 조회
    2. `/icd/entity/{entity_id}` API 호출
    3. 응답에서 `theCode` 또는 해당 엔티티의 코드 필드를 가져와서 표에 기록

### 4. 더 확장해서 200개까지 만들고 싶다면

- 위 표 구조를 그대로 복사해서
    - **부위별 세분화**(예: “요추 1번 압박골절”, “요추 2번 압박골절” 등),
    - **좌/우, 일측/양측**,
    - **급성/만성**
    을 나누면 **쉽게 200개 이상** 확장할 수 있습니다.
- 다만, 그 순간부터는 **기관별/국가별 세부 분류(KCD 등)**와 어긋날 위험이 커지므로
→ 실제 업무용이면 **반드시 공식 코드북을 병행**하셔야 합니다.
